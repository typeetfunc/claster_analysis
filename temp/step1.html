Выбор метрики
Евклидово расстояние ?Для определения расстояния между предприятиями используется формула, которая Представляет собой геометрическое расстояние в многомерном пространстве. Наиболее распространенная функция расстояния.
  • p(x,x') - расстояние между i-тым и j-тым объектами;
  • x - численное значение k-той переменной для i-того объекта;
  • x' - численное значение k-той переменной для j-того объекта;
  • n - количество переменных, которыми описываются объекты.
 (Степень евклидовой метрики)
"Манхэттенская" ?Расстояние городских кварталов (манхэттенское расстояние) Это расстояние является средним разностей по координатам. В большинстве случаев эта мера расстояния приводит к таким же результатам, как и для обычного расстояния Евклида. Однако для этой меры влияние отдельных больших разностей (выбросов) уменьшается (т.к. они не возводятся в квадрат).
  • p(x,x') - расстояние между i-тым и j-тым объектами;
  • x - численное значение k-той переменной для i-того объекта;
  • x' - численное значение k-той переменной для j-того объекта;
  • n - количество переменных, которыми описываются объекты.

Махаланобиса ?Расстояние Махаланобиса отличается от Евклидова тем, что учитывает корреляции между переменными. При выборе данной метрики отпадает необходимость нормировки значиний.

Метод одиночной связи
Метод "дальнего соседа" ?Степень близости оценивается между наиболее близкими объектами этих кластеров.
Метод "ближнего соседа" ?Степень близости оценивается по степени близости между наиболее отдаленными объектами кластеров.
Метод средней связи ?Степень близости оценивается как средняя величина степеней близости между объектами кластеров.
